# 《锦瑟》

**[唐]李商隐**

锦瑟无端五十弦，一弦一柱思华年。

庄生晓梦迷蝴蝶，望帝春心托杜鹃。

沧海月明珠有泪，蓝田日暖玉生烟。

此情可待成追忆，只是当时已惘然。
