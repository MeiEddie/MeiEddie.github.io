1768. 交替合并字符串 ​

给你两个字符串word1和word2。请你从word1开始，通过交替添加字母来合并字符串。如果一个字符串比另一个字符串长，就将多出来的字母追加到合并后字符串的末尾。

返回 合并后的字符串 。

示例1：

输入：word1 = "abc", word2 = "pqr"

输出："apbqcr"

解释：字符串合并情况如下所示：

word1： a b c

word2： p q r

合并后： a p b q c r

示例2：

输入：word1 = "ab", word2 = "pqrs"

输出："apbqrs"

解释：注意，word2比word1长，"rs"需要追加到合并后字符串的末尾。

word1： a b

word2： p q r s

合并后： a p b q r s

示例3：

输入：word1 = "abcd", word2 = "pq"

输出："apbqcd"

解释：注意，word1比word2长，"cd" 需要追加到合并后字符串的末尾。

word1： a b c d

word2： p q

合并后： a p b q c d

提示：

1 <= word1.length, word2.length <= 100

word1和word2由小写英文字母组成

我的尝试

python
class Solution(object):
    def mergeAlternately(self, word1, word2):
        """
        :type word1: str
        :type word2: str
        :rtype: str
        """
        # 这里鉴于我的poor language，我打算使用邪修法
        # 使用[:]来回切割，持续100次，看我操作
        ans = ""
        for i in range(100):
            ans += word1[i:i+1]
            ans += word2[i:i+1]
        return ans

运行时间：3ms

389. 找不同 ​

给定两个字符串s和t，它们只包含小写字母。

字符串t由字符串s随机重排，然后在随机位置添加一个字母。

请找出在 t 中被添加的字母。

示例1：

输入：s = "abcd", t = "abcde"

输出："e"

解释：'e'是那个被添加的字母。

示例2：

输入：s = "", t = "y"

输出："y"

提示：

0 <= s.length <= 1000

t.length == s.length + 1

s 和 t 只包含小写字母

我的尝试

python
class Solution(object):
    def findTheDifference(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: str
        """
        # 暴力方法，我想用count遍历26个单词
        a = 'abcdefghijklmnopqrstuvwxyz'
        for i in range(26):
            num1 = s.count(a[i])
            num2 = t.count(a[i])
            if num1 != num2:
                return a[i]

运行时间：0ms

可以，54个案例全过了，我真厉害

曾经报错：写了int num1 = （省略），Python不支持在变量前写int指定类型，这是Java、C语言的写法，Python变量是动态类型，直接赋值即可。

28. 找出字符串中第一个匹配项的下标 ​

给你两个字符串haystack和needle，请你在haystack字符串中找出needle字符串的第一个匹配项的下标（下标从0开始）。如果needle不是haystack的一部分，则返回-1 。

示例1：

输入：haystack = "sadbutsad", needle = "sad"

输出：0

解释："sad"在下标0和6处匹配。

第一个匹配项的下标是0，所以返回0。

示例2：

输入：haystack = "leetcode", needle = "leeto"

输出：-1

解释："leeto"没有在"leetcode"中出现，所以返回-1。

提示：

1 <= haystack.length, needle.length <= 104

haystack和needle仅由小写英文字符组成

我的尝试

python
class Solution(object):
    def strStr(self, haystack, needle):
        """
        :type haystack: str
        :type needle: str
        :rtype: int
        """
        # 这题太干净了，学过基础就会做
        return haystack.find(needle)

242. 有效的字母异位词 ​

给定两个字符串s和t，编写一个函数来判断t是否是s的字母异位词。

示例1:

输入: s = "anagram", t = "nagaram"

输出: true

示例2:

输入: s = "rat", t = "car"

输出: false

提示:

1 <= s.length, t.length <= 5 * 104

s 和 t 仅包含小写字母

进阶: 如果输入字符串包含unicode字符怎么办？你能否调整你的解法来应对这种情况？

我的尝试

python
class Solution(object):
    def isAnagram(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: bool
        """
        # 和之前的那道找不同一样，count遍历26个单词
        a = 'abcdefghijklmnopqrstuvwxyz'
        for i in range(26):
            b = s.count(a[i])
            c = t.count(a[i])
            if b != c:
                return False
        return True

运行时间：3ms

曾经报错：true/false小写，题目要求输出true/false，只是文字描述输出结果，是通用自然语言写法，不是Python代码语法。力扣等算法题的题目描述，不管什么编程语言，统一用小写true/false表示布尔结果；但不同编程语言有自己的语法规范，不能直接照搬题目里的小写单词写进代码。
