### 1768. 交替合并字符串

给你两个字符串word1和word2。请你从word1开始，通过交替添加字母来合并字符串。如果一个字符串比另一个字符串长，就将多出来的字母追加到合并后字符串的末尾。

返回 合并后的字符串 。

示例1：

输入：word1 = "abc", word2 = "pqr"

输出："apbqcr"

解释：字符串合并情况如下所示：

word1：  a   b   c

word2：    p   q   r

合并后：  a p b q c r

示例2：

输入：word1 = "ab", word2 = "pqrs"

输出："apbqrs"

解释：注意，word2比word1长，"rs"需要追加到合并后字符串的末尾。

word1：  a   b 

word2：    p   q   r   s

合并后：  a p b q   r   s

示例3：

输入：word1 = "abcd", word2 = "pq"

输出："apbqcd"

解释：注意，word1比word2长，"cd" 需要追加到合并后字符串的末尾。

word1：  a   b   c   d

word2：    p   q 

合并后：  a p b q c   d

提示：

1 <= word1.length, word2.length <= 100

word1和word2由小写英文字母组成

**我的尝试**

```python
class Solution(object):
```

```python
class Solution(object):
```

### 459. 重复的子字符串

给定一个非空的字符串s，检查是否可以通过由它的一个子串重复多次构成。

示例1:

输入: s = "abab"

输出: true

解释: 可由子串"ab"重复两次构成。

示例2:

输入: s = "aba"

输出: false

示例3:

输入: s = "abcabcabcabc"

输出: true

解释: 可由子串"abc"重复四次构成。 (或子串"abcabc"重复两次构成。)

提示：

1 <= s.length <= 10^4

s 由小写英文字母组成

**我的尝试**

```python
class Solution(object):
    def repeatedSubstringPattern(self, s):
        """
        :type s: str
        :rtype: bool
        """
        # 我打算先找出第一个重复单元，然后不断检测
        if len(s) == 1:
            return False
        i = 1
        a = s[0]
        b = s[1]
        while a != b:
            if i > len(s)//2+1:
                return False
            a = s[0:i]
            b = s[i:2*i]
            i += 1
        c = len(s)%len(a)
        if c == 0:
            d = len(s)//len(a)
            for e in range(d):
                if s[len(a)*e:len(a)*(e+1)] != a:
                    return False
            return True
        else:
            return False
```

做错了，有一个示例是"abaababaab"，我这个代码会测出来aba，导致输出False，但是abaab是一个正确答案，我这个方向有点错误

> len(str)可以获取字符串str的长度

> 其中错误之一：没有“if len(s) == 1:”的判断，导致后面出错

**题解学习**

来自力扣官方题解

```python
class Solution:
    def repeatedSubstringPattern(self, s: str) -> bool:
        n = len(s)
        for i in range(1, n // 2 + 1):
            if n % i == 0:
                if all(s[j] == s[j - i] for j in range(i, n)):
                    return True
        return False
```

这里n是长度

### 283. 移动零

给定一个数组nums，编写一个函数将所有0移动到数组的末尾，同时保持非零元素的相对顺序。

请注意，必须在不复制数组的情况下原地对数组进行操作。

示例1:

输入: nums = [0,1,0,3,12]

输出: [1,3,12,0,0]

示例2:

输入: nums = [0]

输出: [0]

提示:

1 <= nums.length <= 104

-231 <= nums[i] <= 231 - 1

进阶：你能尽量减少完成的操作次数吗？

**我的尝试**

```python
class Solution(object):
    def moveZeroes(self, nums):
        """
        :type nums: List[int]
        :rtype: None Do not return anything, modify nums in-place instead.
        """
        a = nums.count(0)
        for i in range(a):
            nums.remove(0)
            nums.append(0)
        return nums
```

### 66. 加一

给定一个表示**大整数**的整数数组digits，其中digits[i]是整数的第i位数字。这些数字按从左到右，从最高位到最低位排列。这个大整数不包含任何前导0。

将大整数加1，并返回结果的数字数组。

示例1：

输入：digits = [1,2,3]


输出：[1,2,4]

解释：输入数组表示数字123。

加1后得到 123 + 1 = 124。

因此，结果应该是 [1,2,4]。

示例2：

输入：digits = [4,3,2,1]

输出：[4,3,2,2]

解释：输入数组表示数字4321。

加1后得到 4321 + 1 = 4322。

因此，结果应该是 [4,3,2,2]。

示例3：

输入：digits = [9]

输出：[1,0]

解释：输入数组表示数字9。

加1得到了 9 + 1 = 10。

因此，结果应该是 [1,0]。

提示：

1 <= digits.length <= 100

0 <= digits[i] <= 9

digits不包含任何前导0。

**我的尝试**

```python
class Solution(object):
    def plusOne(self, digits):
        """
        :type digits: List[int]
        :rtype: List[int]
        """
        num = 0
        a = len(digits)
        b = digits[::-1]
        c = []

        for i in range(a):
            num += b[i]*(10**i)

        num += 1
        while num > 0:
            c.append(num%10)
            num //= 10
        
        d = c[::-1]
        return d
```

运行时间：0ms
